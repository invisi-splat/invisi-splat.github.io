Website In Website

Version 0.1.0

Enter Website here:


This is a website that sends a GET request and renders it on the page.

It bypasses the same origin policy (currently) through cors-anywhere.

This means that you can view a website... within a website!

Note that this is still experimental and many features are yet to be added.

A lot of things still don't work. But hey, you can view Reddit.
