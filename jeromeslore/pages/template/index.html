Jerome's

Contributors: invisi., FreakishFool
For the mascot, see Jerome.

Jerome's is the name of and a prefix for several entities related to Jerome's Server and people in it. It is often used in combination with some other descriptive word to show an association with the server group, for example, Jerome's Anarchy, and Jerome's Groovy.

The Discord server, when there are no active server-wide Minecraft worlds in progress, defaults to this name.
